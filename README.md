# Form Validator (JavaScript)

A modern form validation project built with vanilla JavaScript.

---

## 🚀 Features

* Real-time validation (on input & blur)
* Name & surname validation (Ukrainian + English letters)
* Email validation using regex
* Phone number masking with IMask
* Visual feedback (valid / invalid states)
* Accessible form (ARIA attributes)
* Clean and responsive UI

---

## 🛠 Technologies

* HTML5
* SCSS (BEM methodology)
* JavaScript (ES6+)
* IMask

---

## 📱 Fields

* First Name
* Last Name
* Email
* Phone (with mask)

---

## ⚙️ Validation Rules

* **Names:** only letters (UA/EN), minimum length validation
* **Email:** standard format
* **Phone:** required full mask input
